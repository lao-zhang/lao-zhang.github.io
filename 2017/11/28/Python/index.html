python笔记

Python

1.basic

数据存储：

list，有序，元素可调节列表，符号为[ ]；

tuple，有序，元素不可调节列表（所有元素需要初始化，且之后不可变动），符号为( )，

​ 只有一个元素（假设为1）的tuple也要写成（1，）；

dict，字典，key-value存储，查询速度快。（类似java-web的setAttribute）。

Set,只存储一组key值，无序，自动过滤重复值，创建时需要一个list作为输入集合。

逻辑控制语句：

python里逻辑语句没有括号，而是采用了“：”和首行缩进，这点要注意。

if：else：

for…in…：

while:

break;

continue;
